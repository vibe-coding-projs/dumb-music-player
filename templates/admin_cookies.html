מה זה Cookies ולמה צריך אותם?

YouTube Cookies מאפשרים להוריד שירים שמוגבלים גיאוגרפית או דורשים אימות.

אפשרות 1 — העתקה מ-Chrome DevTools (ללא תוסף, מומלץ!)

1
פתח youtube.com בדפדפן Chrome, לחץ על המקש F12 לפתיחת DevTools, ולחץ על הלשונית Network (רשת) בחלק העליון של הכלים.
2
בסרגל הפילטרים שמתחת ללשוניות, לחץ על Doc (קיצור של Document). זה מסנן ומציג רק את הבקשה הראשית של הדף — רק בה יש cookies.
אם אינך רואה "Doc" — חפש את הכפתורים: All / Fetch/XHR / JS / CSS / Img / Doc
3
לחץ F5 לרענון הדף. תראה בקשה אחת בשם www.youtube.com — לחץ עליה.
4
בפאנל שנפתח מימין, לחץ על הלשונית Headers. גלול מטה עד לחלק Request Headers. תראה שורה שמתחילה ב־cookie: עם ערך ארוך מאוד.
אין שורת cookie? ודא שאתה מחובר ל-YouTube לפני הרענון.
5
לחץ לחיצה ימנית על הערך שליד cookie: ← בחר Copy value (העתק ערך).
⚠️ העתק רק את הערך — לא את המילה "cookie:" עצמה.
✓
הדבק את הערך שהעתקת בתיבת הטקסט למטה ולחץ שמור. המערכת תמיר אוטומטית לפורמט הנכון.
הערך שהעתקת נראה כך: YSC=abc123; VISITOR_INFO1_LIVE=xyz; SID=...
⚠️ שימו לב: יש להעתיק את הערך מלשונית Network → Headers בלבד (שיטה מעלה). שיטות אחרות כמו document.cookie בקונסול לא עובדות — הן מפספסות את ה-cookies החשובים ביותר (httpOnly) שיוטיוב משתמש בהם לאימות.

אפשרות 2 — פורמט Netscape (cookies.txt)

  1. התקן את התוסף "Get cookies.txt LOCALLY" ב-Chrome/Firefox
  2. היכנס ל-YouTube בדפדפן
  3. לחץ על אייקון התוסף בעת שאתה באתר YouTube ולחץ "Export"
  4. הדבק את התוכן בתיבת הטקסט למטה

קישורים לתוספים:
• Chrome Web Store
• Firefox Add-ons
