מה זה Cookies ולמה צריך אותם?

YouTube Cookies מאפשרים להוריד שירים שמוגבלים גיאוגרפית או דורשים אימות.

אפשרות 1 — Cookie header גולמי (ללא תוסף, מומלץ!)

  1. פתח YouTube בדפדפן → לחץ F12 → לשונית Network
  2. רענן את הדף ולחץ על כל בקשה ל-youtube.com
  3. תחת Request Headers → מצא את השורה Cookie:
  4. העתק את הערך בלבד (ה-key=value; key2=value2...)
  5. הדבק בתיבת הטקסט למטה — המערכת תמיר אוטומטית לפורמט הנכון

אפשרות 2 — פורמט Netscape (cookies.txt)

  1. התקן את התוסף "Get cookies.txt LOCALLY" ב-Chrome/Firefox
  2. היכנס ל-YouTube בדפדפן
  3. לחץ על אייקון התוסף בעת שאתה באתר YouTube ולחץ "Export"
  4. הדבק את התוכן בתיבת הטקסט למטה

קישורים לתוספים:
• Chrome Web Store
• Firefox Add-ons
